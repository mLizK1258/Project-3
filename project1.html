Introduction

This is a group assignment for creating company website, my partner is Yenan Huang. We have 4 weeks to completed the project and we used HTML and CSS to create this website. I learn a lot about creating a website and website designing. I will be going through my process of designing this website.

Week 1

In the first week, me and my partner create a simple layout on Figma to help us visualize the layout for the website. Our target for the website is a young people, so we tried to do the "clean" layout as much as we can with the simple design but are still very attractive.

After we finished the layout, me and Yenan started doing our own part for each page of the website. In the first week, I completed the layout About page for the company, and adding some images for partnerships.
